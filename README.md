# gitee-calendar
this will creep gitee calendar and return a array of data

## 简介
你可以通过部署这个Python项目到vercel或你自己的服务器来爬取你gitee的提交记录

## 使用方法
你可以通过fork这个库并部署到vercel的方法来免费托管这个项目
部署到vercel的方法在此不多赘述了

或者你也可以通过修改demo链接中的参数来得到自己的gitee数据

用户名是你git主页中url中的参数
